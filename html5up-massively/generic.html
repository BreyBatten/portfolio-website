H3rra

Wedding planning application where couples can keep track of all their important wedding info. Built with React on the front end, NodeJS and Express on the back end on a PostgreSQL database.

  • Project was taken from initial user-research, competition research and planning to production
  • Utilizes automated email invites that guests receive when a couple adds them to their guest list. Couple's list automatically updates after guests fill out form in email.
  • Team was able to reach MVP in given time-frame. Additional features, such as automated emails were added to project prior to deadline.

TP Trader

Full-Stack application for trading items, was specifically built for the toilet paper shortage due to the Covid-19 pandemic. Front end is built with React, back end is built with NodeJS and Express and hosted on a PostgreSQL database. Project is currently being completed.

  • I was the back end developer for this application.
  • Team of 3 developers were able to build functioning Full-Stack application in 12 days, working part-time.
